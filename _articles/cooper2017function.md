abstract: |
  In this article, the author shares models that were created for class
  projects in a section of College Algebra taught as part of a first-year
  cohort program in which the participating students complete eight
  introductory college courses as a learning community. Using real-world data
  and GeoGebra function fitting commands, the author has developed models
  related to a community engagement project theme of drug and alcohol
  awareness. Quadratic, rational, exponential, and logistic models are
  presented along with data and a discussion of how GeoGebra was used to
  create the models.

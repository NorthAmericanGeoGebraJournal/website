  In this article, the author demonstrates how GeoGebra can be used to build
  fraction bar models for conceptualizing equivalent fractions as well as
  addition, subtraction, multiplication, and division of fractions. Each
  construction uses checkboxes and input boxes to allow users to dynamically
  interact with the constructions and sequences of polygons to create the
  fraction bars.

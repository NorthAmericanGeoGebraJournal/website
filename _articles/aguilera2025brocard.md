abstract: |
  The Brocard circle (or seven-point circle) is a circle derived from a given
  triangle. There have been several discoveries regarding the Brocard circle.
  In this study, a compilation of existing properties of the Brocard Circle
  has been presented, along with properties not currently found. These
  properties were discovered using GeoGebra.
